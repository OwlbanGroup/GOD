# Project Run Checklist for GOD-TOKEN-COIN and OSCAR-BROOME-REVENUE

## GOD-TOKEN-COIN (Hardhat)
- [ ] Ensure dependencies: cd GOD-TOKEN-COIN && npm install
- [ ] Compile: cd GOD-TOKEN-COIN; npx hardhat compile
- [ ] Test: cd GOD-TOKEN-COIN; npm test  (runs hardhat test)

## OSCAR-BROOME-REVENUE (TypeScript/Jest)
- [ ] Ensure dependencies: cd OSCAR-BROOME-REVENUE && npm install
- [ ] Type check: cd OSCAR-BROOME-REVENUE; npx tsc --noEmit
- [ ] Test: cd OSCAR-BROOME-REVENUE; npm test  (runs jest)

## PowerShell Note
Use `;` for command chaining in PowerShell 5.1: cd dir; command
Or run separate commands. `&&` requires PowerShell 7+.
